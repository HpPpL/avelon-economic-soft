# Demo data

This directory contains small synthetic examples for portfolio review only. The values are invented and are not copied from any client, company, reservoir, contract, report, or internal model.

`synthetic_project_input.csv` mirrors the type of yearly production, price, CAPEX, OPEX, and tax assumptions the historical spreadsheet model consumed. The archived Python code originally expected Excel workbooks with a specific multi-sheet layout; the CSV is included as safe documentation of the data shape rather than a drop-in replacement for the removed client workbooks.
